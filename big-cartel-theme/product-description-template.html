About the Design

Simple and understated, this "Code" design is for the code rockstar in you. Rock it to the club, rock it when you skate, rock it when you wake up and code....whatever you do just rock it.

About the Hoodie

This zippered hoodie is charcoal gray (nearly black) and printed on an Independent Trading AFX4000Z. The size is custom printed inside and the regular sizing tag has been removed for your comfort.

Garment Fit

The Code Hoodie is a slim fit garment, and almost runs a full size small, so please keep that in mind when ordering. For actual specs on sizing please refer to this sizing chart when choosing your size.
